# eRoyal Final Year Project Penaflex Prompt

Create a print-ready vertical penaflex poster for a university final year project exhibition.

## Print Size And Quality
- Final size: 2.5 feet width x 3 feet height.
- Exact design size: 30 inches wide x 36 inches high, portrait orientation.
- Export as high-quality PDF for printing.
- Use 300 DPI or vector-safe output where possible.
- Keep all important text and logos inside a safe margin of at least 0.45 inch.
- Use a clean white or very light background, with premium green, teal, slate, blue, and amber accents.
- Avoid too much paragraph text. The poster should explain the full project through short headings, small points, icons, and diagrams.

## Main Text
- Top heading: FINAL YEAR PROJECT
- Project title below it: eRoyal Housing Society Management System
- Short tagline: A role-based digital platform for modern society operations.
- University logo: Place The University of Lahore logo at the top left.
- Project/app logo: Place the eRoyal app logo at the top right.

## Visual Style
- Formal university exhibition design, not a marketing flyer.
- Use modern dashboard-style layout with clean lines and readable typography.
- Use the project brand green/teal as the main accent, but include blue, amber, and slate so the design does not look single-color.
- Use rounded rectangles only lightly, with small border radius.
- Use thin separators, subtle shadows, and organized sections.
- Keep all text large enough for penaflex printing and viewing from a few feet away.
- Do not overcrowd the poster. Prefer short labels and diagrams over long explanations.
