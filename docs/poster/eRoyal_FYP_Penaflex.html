eRoyal Mobile + Web App

eRoyal Housing Society Management System

A role-based digital platform for modern society operations.

eRoyal centralizes residents, bills, complaints, announcements, marketplace listings, vehicle registration, gate entry logs, image uploads, and AI-assisted support in one secure Firebase-powered application.

1

One Platform, Three Protected Portals

Admin Portal

  • Create and manage society users
  • Generate monthly resident bills
  • Verify uploaded payment proofs
  • Resolve complaints and monitor logs

Resident App

  • View announcements and bills
  • Upload payment proof images
  • Submit complaints with photos
  • Post marketplace and vehicle details

Security Gate

  • Record entry and exit activity
  • Track active vehicles at gate
  • Handle residents, visitors, services
  • Use OCR for license plate scanning

Core Project Modules

UUser And Roles

Admin manages residents, admins, and security staff with authenticated role-based access.

BBilling

Monthly bills are generated, sent to residents, and verified using uploaded payment proof.

CComplaints

Residents submit complaints with photos, while admin tracks status and resolution notes.

AAnnouncements

Society notices are published digitally so residents can receive updates from one place.

MMarketplace

Residents can create property sale or rent listings with descriptions and photos.

VVehicle Gate

Security records resident, visitor, and service vehicle entries with live active status.

AIAssistant + OCR

Gemini supports resident assistance and camera-based license plate recognition.

FFirebase Cloud

Authentication, Firestore, Storage, security rules, and real-time data updates power the app.
